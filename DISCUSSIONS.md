# API NODE Discussions Guide

Use GitHub Discussions for questions, guide requests, working configs, and compatibility feedback.

## Good Discussion Topics

- Request a guide for a tool, SDK, framework, or coding agent.
- Share a redacted config that works with API NODE.
- Ask for help with base URL, model, Responses API, or streaming setup.
- Suggest a small docs/example PR for an OpenAI-compatible client.

## Before Posting

- Remove API keys, tokens, account IDs, private URLs, and billing details.
- Include the tool name and version when relevant.
- Include the base URL and model ID if they are safe to share:

```text
Base URL: https://apinode.pro
Model: gpt-5.5
```

## Suggested Categories

- Guide requests
- Working configs
- Compatibility help
- Announcements

## Maintainer Response Style

- Help with concrete config fixes.
- Link to the closest existing guide.
- Ask for redacted config only when needed.
- Do not request public promotion or README placement.
