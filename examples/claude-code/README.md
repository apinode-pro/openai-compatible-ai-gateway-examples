# API NODE Claude Code Setup

Use API NODE with OpenAI-compatible clients, wrappers, or proxy setups that accept a custom `base_url`.

Recommended values:

```text
API key:  APINODE_API_KEY
Base URL: https://apinode.pro
Model:    your selected model
```

Suggested smoke test prompt:

```text
Inspect this repository and summarize the project in three bullets.
```

Keep API keys in environment variables or your local secret manager. Do not commit API keys to GitHub.
