# API NODE Cursor Setup

Use API NODE anywhere Cursor accepts an OpenAI-compatible custom API endpoint.

Recommended values:

```text
API Key:  your API NODE key
Base URL: https://apinode.pro
Model:    gpt-5.5 or another model available in your API NODE account
```

After saving the configuration, send a small test prompt:

```text
Reply with "API NODE is connected" and nothing else.
```

If the request fails, verify:

- The API key has no extra spaces.
- The base URL is `https://apinode.pro` unless your account uses a custom endpoint.
- The selected model is enabled for your account.
