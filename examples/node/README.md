# API NODE Node.js Example

Install dependencies:

```bash
npm install
```

Run the example:

```bash
export APINODE_API_KEY="your_api_key"
export APINODE_BASE_URL="https://apinode.pro"
export APINODE_MODEL="gpt-5.5"
npm start
```

The example uses the official OpenAI Node SDK with a custom `baseURL` and the Responses API.
