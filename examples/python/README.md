# API NODE Python Example

Install dependencies:

```bash
python -m pip install -r requirements.txt
```

Run the example:

```bash
export APINODE_API_KEY="your_api_key"
export APINODE_BASE_URL="https://apinode.pro"
export APINODE_MODEL="gpt-5.5"
python main.py
```

The example uses the official OpenAI Python SDK with a custom `base_url` and the Responses API.
