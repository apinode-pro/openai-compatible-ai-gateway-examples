# Use Cline With API NODE

Cline can work with OpenAI-compatible model providers for coding-agent workflows.

API NODE endpoint:

```text
https://apinode.pro
```

## Recommended Values

```text
Provider: OpenAI-compatible
API key:  your API NODE key
Base URL: https://apinode.pro
Model:    gpt-5.5
```

## Smoke Test

Ask Cline:

```text
Create a small README section that explains how to run this project locally.
```

## Notes

- Keep your API key out of project files.
- Use a small task first to verify credentials and endpoint behavior.
- If your Cline version separates model ID and base URL fields, keep `gpt-5.5` as the model and `https://apinode.pro` as the compatible endpoint.
