# Use Roo Code With API NODE

Roo Code supports configurable model providers for editor-based coding workflows.

API NODE endpoint:

```text
https://apinode.pro
```

## Recommended Values

```text
Provider: OpenAI-compatible
API key:  your API NODE key
Base URL: https://apinode.pro
Model:    gpt-5.5
```

## Smoke Test

Ask Roo Code:

```text
Find the main entry point of this project and explain it in two paragraphs.
```

## Notes

- Store API keys in your editor or OS secret storage, not in the repository.
- Use a model enabled for your API NODE account.
- Start with read-only tasks before enabling file edits.
